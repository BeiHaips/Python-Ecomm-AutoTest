 移动端商城全栈开发项目 (Vue.js + MySQL)

 项目定位
本项目是基于 Vue.js 开发的响应式移动端商城原型。实现了从前端 UI 组件化渲染到后端 MySQL 数据库映射的全链路流程，旨在验证 Web 前端开发（中级）证书中的核心实战技能。

 技术实现
* 前端: 采用 Vue.js 指令与组件化思维，实现商品列表动态渲染。
* 后端: 利用 Python 编写逻辑连接 MySQL，确保数据持久化。
* 排版: 针对移动端屏幕进行像素级响应式适配。

 核心目录说明
* `/src`: Vue 业务组件与前端逻辑。
* `/data`: MySQL 数据库初始化 SQL 脚本。
